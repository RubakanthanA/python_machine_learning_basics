# Fit the regression model
# führt eine lineare Regression durch, um eine Beziehung zwischen den Trainingsdaten X_train (hier die Fläche) und y_train (hier der Preis) zu finden, und gibt die Ergebnisse der Regression aus. 

# stats.linregress: führt eine lineare Regression durch
# X_train: Die Eingabedaten (hier die Fläche) der Trainingsdaten.
# y_train: Die Zielwerte (hier der Preis) der Trainingsdaten.


slope, intercept, r, p, std_err = stats.linregress(X_train, y_train)

# Print results of the regression model - Ergebnisse der Regression ausgeben
# Die Ergebnisse der Regression werden formatiert und ausgegeben:
# - Intercept: Der Schnittpunkt der Regressionslinie mit der y-Achse. Ein positiver oder negativer Intercept bedeutet, dass bei einer Fläche von Null ein bestimmter Grundpreis existiert.
# - Slope: Die Steigung der Regressionslinie, die anzeigt, wie stark der Preis bei einer Veränderung der Fläche ansteigt.
# - p-Wert: Ein p-Wert unter 0,05 zeigt oft eine statistisch signifikante Beziehung zwischen X und y.
# - R-squared: Das Bestimmtheitsmass, auch als 𝑅2 bekannt, gibt an, wie viel der Varianz in y durch X erklärt wird. 
#       Ein höheres R2 (nahe 1) deutet auf eine stärkere Übereinstimmung der Regressionslinie mit den Daten hin.

print('Linear regression result:')
print(f'Intercept with y-axis (alpha):            {intercept:.2f}')
print(f'Slope of regression line (beta):          {slope:.3f}')
print(f'p-value:                                  {p:.4f}')
print(f'R-squared (coefficient of determination): {r**2:.4f}')

# y_train.describe() gibt eine statistische Zusammenfassung der Zielwerte y_train, inkl.Mittelwerts, Standardabweichung, Minimums, Maximums und R2.
y_train.describe()

Plot regression line¶

In [8]:
# Function to calculate model predictions
# myfunc(x) berechnet die Vorhersage für einen gegebenen Wert x basierend auf dem linearen Regressionsmodell.


def myfunc(x):
    return slope * x + intercept

# Apply myfunc() to x, i.e. make predictions - Vorhersagen auf die Trainingsdaten anwenden:
# map(myfunc, X_train): Funktion myfunc wird auf jeden Wert in X_train angewendet, um die Vorhersagen für die Trainingsdaten zu berechnen.
# pd.Series(...): Die Ergebnisse werden in eine pandas.Series umgewandelt und als mymodel gespeichert, sodass sie für die Visualisierung verwendet werden können.

mymodel = pd.Series(map(myfunc, X_train))

# Scatterplot with regression line
# plt.scatter(X_train, y_train, s=10, color='green'): Erstellt ein Streudiagramm der tatsächlichen Werte in X_train und y_train.
# X_train: Flächenwerte (Merkmal).
# y_train: Tatsächliche Preise (Zielvariable).
# s=10: Bestimmt die Größe der Punkte.

# plt.plot(X_train, mymodel, color='darkred', linestyle='dashed'): Zeichnet die Regressionslinie basierend auf den Vorhersagen mymodel
# linestyle='dashed': Die Linie wird gestrichelt dargestellt.

plt.figure(figsize=(6,4))
plt.scatter(X_train, y_train, s=10, color='green')
plt.plot(X_train, mymodel, color='darkred', linestyle='dashed')
plt.title('Simple Linear Regression')
plt.xlabel('area (m2)')
plt.ylabel('price (CHF)')

plt.show()

# Calculate model residuals for train data - Berechnung der Modellresiduen für die Trainingsdaten:
# residuals: Ein Residuum (oder Fehler) ist der Unterschied zwischen tatsächlichen Wert (y_train) und dem vom Modell vorhergesagten Wert (mymodel).
# Residuum beschreibt, wie weit die Modellvorhersage vom tatsächlichen Wert abweicht.
#  - positives (unterschätzt) Residuum bedeutet, dass die tatsächlichen Werte höher sind als die Vorhersage
#  - ein negatives (überschätzt), dass sie niedriger sind.

residuals = y_train - mymodel

# Check the first residual value in our data set
# mymodel[0]: Dies ist die erste Vorhersage des Modells in mymodel. {mymodel[0]:.2f} auf zwei Dezimalstellen gerundet.
# y_train[0]: Dies ist der erste tatsächliche Preis in y_train, auf zwei Dezimalstellen gerundet.
# residuals[0]: Das erste Residuum, Unterschied zwischen der ersten Modellvorhersage und dem ersten tatsächlichen Wert darstellt.
# - über- oder unterschätzt es systematisch die Werte

# BLEIBT IMMER GLEICH
print(f'1st Predicted price in dataset: {mymodel[0]:.2f}')
print(f'1st Observed price in dataset: {y_train[0]:.2f}')
print(f'1st Residual price in dataset: {residuals[0]:.2f}')

# Add constant to the model - multiples Regressionsmodell, führt es aus und zeigt die Zusammenfassung der Ergebnisse.
# Konstante oder einen Interzeptor (Achsenabschnitt) hinzuzufügen, Basislinie des Modells 
# dd_constant von statsmodels (sm) fügt eine Spalte mit Einsen zu den Eingabedaten X2_train hinzu. Diese Spalte repräsentiert die Konstante, die das Modell benötigt, um die Achsenabschnitts-Koeffizienten (den konstanten Wert in der Regression) zu berechnen.
# X2_train_const, die X2_train plus eine zusätzliche Spalte für die Konstante enthält.

X2_train_const = sm.add_constant(X2_train)

# Create the multiple regression model
# Modell erstellen,
olsmod = sm.OLS(y_train, X2_train_const)
olsres = olsmod.fit()

# Print full model output
# Ergebniss anzeigen
print(olsres.summary())

# Koeffizienten (Coefficients): Die Werte für die Konstante und die Regressionskoeffizienten der Prädiktorvariablen.

# R² und Adjusted R²: Maße für die Erklärungsfähigkeit des Modells.
#   -R² gibt an, wie gut die unabhängigen Variablen die Varianz der Zielvariable erklären.
# F-Statistik: Zeigt, ob das Modell als Ganzes signifikant ist.

Interpretation of the relevant (in this course) statistics in the table above¶

R-squared: This is the coefficient of determination (see slides of lessons). A value of 0.522 means, that the explanatory variables explain 52% of the variaton of our target variable (rental prices) - not bad, but could be improved.

coef: These are the estimated coefficients of the explanatory variables ('slopes of the regression line' of each variable). These are nedded for the price predictions in our model.

P>|t|: These are the p-values. If < 0.05, the explanatory variables shows a statistically siginificant (5% significance level) contribution in explaining the target variable. Except for the distiance to the nearest supermarkt, all variables are significant here.
