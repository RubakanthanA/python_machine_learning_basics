Simple Linear Regression: Confidence- and Prediction-Intervals

This notebook demonstrates how to create confidence and prediction intervals for a linear regression model using Python, based on rental price data (Y) and living area (X).

See also: https://lmc2179.github.io/posts/confidence_prediction.html

Generate Apartment Data

We will use the provided data representing rental prices (Y) and living area (X).

In[]:
# Set the random seed for reproducibility - imulierte Datenmenge fr rental_price und living_area
# Zufalls-Seed fr Reproduzierbarkeit setzen, zufllig generierte Werte sind dann immer gleich
np.random.seed(42)

# Generate values for rental_price and living_area
# np.linspace(45, 160, 25) erzeugt 25 gleichmig verteilte Werte zwischen 45 und 160
# - Das Ergebnis ist eine Liste von 25 Werten, die x zugewiesen wird.

# np.interp(x, [45, 160], [1500, 4900]) interpoliert die Werte in x
# - Wenn Wohnflche 45 dann Mietpreis 1500
# - Wenn Wohnflche 160 dann Mietpreis von 4900 
#   Fr Werte dazwischen wird  Mietpreis proportional zwischen 1500 und 4900 skaliert
#   - Es entsteht eine lineare Beziehung zwischen living_area und rental_price

# np.random.normal(0, 400, 25), fgt dem Mietpreis eine Zufallsabweichung hinzu
# - die aus einer Normalverteilung mit Mittelwert 0 und Standardabweichung 400 stammt.
# - Das Ergebnis ist eine Liste von 25 Werten, die y zugewiesen wird.

x = np.linspace(45, 160, 25)
y = np.interp(x, [45, 160], [1500, 4900]) + np.random.normal(0, 400, 25)

# Create a new DataFrame with 25 data points
# pd.DataFrame(...) erstellt DataFrame mit zwei Spalten: rental_price (y) und living_area (x).
data = pd.DataFrame({
    'rental_price': y,
    'living_area': x
})

df = pd.DataFrame(data)
df.columns = ['rental_price', 'living_area']
df.head()

Fit an OLS Regression Model

We will use statsmodels to fit an ordinary least squares (OLS) regression model for rental price as a function of living area.

In[]:
# Fit OLS regression
# Der String 'rental_price ~ living_area' spezifiziert die Formel,
# rental_price die Zielvariable (abhngige Variable)
# living_area die Prdiktorvariable (unabhngige Variable) ist.
model = smf.ols('rental_price ~ living_area', data=df)

# Get and show results
# model.fit() passt das Modell an die Daten an und berechnet die optimalen Koeffizienten fr die Regressionsgleichung.
# results.summary() gibt eine detaillierte Zusammenfassung der Regressionsanalyse aus, einschlielich:
# - Koeffizienten: Schtzwerte fr den Achsenabschnitt und die Steigung.
# - p-Werte und R-Wert: Mazahlen zur Signifikanz und Gte des Modells.
results = model.fit()
print(results.summary())

# Plot data and regression line - Regressionsgleichung als Text hinzufgen:
# plt.scatter(df['living_area'], df['rental_price']) erstellt einen Scatter-Plot der Daten, living_area auf der x-Achse und rental_price auf der y-Achse liegt.
# plt.plot(...) zeichnet die Regressionslinie basierend auf den Vorhersagen des Modells (results.predict(df['living_area'])
# linewidth=2 erhht die Linienbreite auf 2.

plt.scatter(df['living_area'], df['rental_price'])
plt.plot(df['living_area'], 
         results.predict(df['living_area']), 
         color='darkred', 
         linewidth=2)
plt.xlabel('Living Area')
plt.ylabel('Rental Price')
plt.title('Living Area vs Rental Price')

# Add annotation (regression equation with f-string)
# plt.text(50, 4500, ...) platziert eine Textannotation im Plot bei den Koordinaten (50, 4500).
# - 50 die Position auf der x-Achso, bei der der Text erscheinen soll.
# - 4500 die Position auf der y-Achse, bei der der Text erscheinen soll.
# :.2f auf zwei Dezimalstellen gerundet.
# fontsize=12 Schriftgre 12

plt.text(50, 4500, f"y = {results.params[0]:.2f} + {results.params[1]:.2f}x", 
         fontsize=12, color='black')

plt.show()

Generate Prediction- and Confidence Intervals

We calculate the prediction- and confidence intervals for the model.

In[]:
# Generate predictions and confidence intervals
# alpha = 0.05, was einem Konfidenzniveau von 95 % entspricht (da 1 - alpha = 0.95, bei bsp. 99% wre alpha = 0.01).
# - Ein Konfidenzintervall von 95 % bedeutet, dass wir zu 95 % sicher sind, dass der wahre Wert innerhalb dieses Intervalls liegt.
# Die Spalten im resultierenden DataFrame predictions enthalten u. a.:
# - mean: Die vorhergesagten Mittelwerte (also die zentralen Schtzwerte fr rental_price).
# - mean_ci_lower und mean_ci_upper: Die untere und obere Grenze des 95 %-Konfidenzintervalls fr den Mittelwert.
# - obs_ci_lower und obs_ci_upper: Die untere und obere Grenze des 95 %-Vorhersageintervalls fr einzelne Beobachtungen.

# STRUKTUR BLEIBT GLEICH NUR ALPHA NDER SICH

alpha = 0.05
predictions = results.get_prediction(df).summary_frame(alpha)

Visualizing the Regression Line and Confidence Interval

We now plot the observed data along with the regression line and confidence intervals.

In[]:
# Plot observed data and regression line
# plt.scatter erzeugt Scatter-Plot der tatschlichen Daten (df['living_area'] und df['rental_price'])
# -um die Beziehung zwischen Wohnflche und Mietpreis darzustellen
# marker='x' legt das Symbol fest
plt.scatter(df['living_area'], 
            df['rental_price'], 
            label='Observed', 
            marker='x', 
            color='black')

# Plot regression line
# plt.plot zeichnet die Regressionslinie.
# df['living_area'] auf X Achse , Wohnflche
# predictions['mean']: als y Achse,  vorhergesagten Mietpreise
# label='Regression line' fgt die Beschriftung fr die Regressionslinie in die Legende ein.
plt.plot(df['living_area'], 
         predictions['mean'], 
         label='Regression line')

# Plot confidence intervals
# plt.fill_between erzeugt schattierten Bereich zwischen den Grenzen des Konfidenzintervalls.
# predictions['mean_ci_lower'] und predictions['mean_ci_upper']: unteren und oberen Grenzen des 95%-Konfidenzintervalls fr die Regressionsvorhersagen.
# alpha=0.4: Transparenz (40 %) des schattierten Bereichs fest, sodass der Bereich halbtransparent ist und die Datenpunkte sowie die Regressionslinie noch sichtbar sind.
# label='95% Confidence Interval' fgt die Beschriftung fr das Konfidenzintervall zur Legende hinzu.
plt.fill_between(df['living_area'], 
                 predictions['mean_ci_lower'], 
                 predictions['mean_ci_upper'], 
                 color='blue', 
                 alpha=0.4, 
                 label='95% Confidence Interval')

# Add legend and labels
plt.xlabel('Living Area (m2)')
plt.ylabel('Rental Price (CHF)')
plt.legend()
plt.show()

Visualizing the Regression Line and Prediction Interval

We now plot the observed data along with the regression line, confidence- and prediction intervals.

# Plot observed data and regression line
# plt.scatter erstellt Streudiagramm der tatschlichen Daten (living_area und rental_price),
# - um die Beziehung zwischen Wohnflche und Mietpreis darzustellen
# marker='x' zeigt als "x"-Symbole

plt.scatter(df['living_area'], 
            df['rental_price'], 
            label='Observed', 
            marker='x', 
            color='black')

# Plot regression line
# plt.plot zeichnet die Regressionslinie.
# predictions['mean']: zentralen Schtzungen der Mietpreise fr die Wohnflchen in df['living_area'].
#label='Regression line' fgt eine Beschriftung fr die Regressionslinie in die Legende ein.
plt.plot(df['living_area'], 
         predictions['mean'], 
         label='Regression line')

# Plot prediction interval
# plt.fill_between: erzeugt schattierten Bereich fr das Vorhersageintervall, der die Unsicherheit fr zuknftige Einzelbeobachtungen darstellt.
# predictions['obs_ci_lower'] und predictions['obs_ci_upper'] sind die unteren und oberen Grenzen des 95 %-Vorhersageintervalls fr die Mietpreise.
#alpha=.5: die Transparenz
plt.fill_between(df['living_area'], 
                 predictions['obs_ci_lower'], 
                 predictions['obs_ci_upper'], 
                 alpha=.5, 
                 label='95% Prediction interval',
                 color='orange')

# Plot confidence interval
# plt.fill_between erzeugt einen schattierten Bereich fr das Konfidenzintervall der Regressionsvorhersagen.
# predictions['mean_ci_lower'] und predictions['mean_ci_upper'] sind die unteren und oberen Grenzen des 95 %-Konfidenzintervalls fr den Mittelwert der Mietpreise.
#alpha=.4: die Transparenz
plt.fill_between(df['living_area'], 
                 predictions['mean_ci_lower'], 
                 predictions['mean_ci_upper'], 
                 alpha=.4, 
                 label='95% Confidence interval',
                 color='blue')

# Add labels
# Beschriftung
plt.xlabel('Living area (m2)')
plt.ylabel('Rental price (CHF)')
plt.legend()

# Show plot
plt.show()

# Unterschied zwischen Konfidenz- und Vorhersageintervall:
#   Konfidenzintervall (blauer Bereich): Gibt den Bereich an, in dem der wahre Mittelwert der Zielvariablen fr einen
#   gegebenen Prdiktorwert (Wohnflche) mit 95 %-iger Wahrscheinlichkeit liegt.
#   Es zeigt die Unsicherheit des Modells in Bezug auf den Mittelwert.
# Vorhersageintervall (oranger Bereich): Gibt den Bereich an, in dem ein zuknftiger einzelner Wert fr die Zielvariable liegen wird.
#   Da es die Variabilitt einzelner Beobachtungen bercksichtigt, ist es breiter als das Konfidenzintervall.
